Tags em HTML

HTML é uma linguagem de marcação de hipertexto, que é utilizada para estruturar páginas web. As tags são elementos que compõem a estrutura de um documento HTML. Elas são delimitadas por caracteres de menor e maior e podem conter atributos que modificam o comportamento ou a aparência do elemento.
